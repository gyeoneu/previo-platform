🔌 COM · 통신 이벤트
DMS-실내기 간 통신 상태. 주소 충돌은 설치기사 임의 설정이 주원인으로 재발성 높음. COM-01(미연결)은 입주 미완료로 운영이슈 성격.
COM
Communication
DMS ↔ 실내기/실외기 통신 상태 이벤트
코드이벤트명발생 조건현장 메모점수 delta추천 액션
COM-01 미연결 호실 실내기 DMS 미등록 / 통신 미연결
1개: -2pt3개↑: -5pt		 입주 미완료 가능성 / 운영이슈 분류 -2~5 모니터링보고서
COM-02 주소 충돌 동일 계통 내 중복 주소 설정
최초: -8pt재발: -12pt		 설치기사 임의 설정이 주원인 / 재발성 주의 -8~12 현장 점검일정 생성
COM-03 F1/F2 배선 이상 통신선 단선/오결선 감지 공사 후 배선 확인 필요 -7 현장 점검
COM-04 통신 PCB 이상 실내기 통신 보드 불량 의심 부품 교체 연계 필요 -10 긴급 점검부품 준비
COM-05 서브실외기 통신 이상 Master↔Slave 실외기 간 통신 단절 DMS 2.5 마스터/슬레이브 연동 확인 -12 긴급 점검모니터링
COM-06 통신 에러 반복 30일 내 동일 계통 통신 에러 3회↑ 구조적 문제 의심 -15 긴급 점검보고서
COM-07 공사 후 통신 이상 입주공사 완료 후 48h 내 통신 오류 발생 설치 보완 연계 처리 -8 현장 점검A/S 일정
❄️ REF · 냉매 이벤트
냉매 누설 우선순위: ①충전포트/패킹(2020년식 6년차 실제 이력) → ②내부 밸브 → ③공사 후 밸브 주변 → ④실내기 연결부(1건) → ⑤배관 본관(질소 기밀테스트로 가능성 낮음).
REF
Refrigerant
냉매 상태 · 누설 · 충전 관련 이벤트 (하절기 가중치 ×1.5)
코드이벤트명발생 조건현장 메모점수 delta추천 액션
REF-01 실내기 추가 후 냉매 부족 실내기 신규 설치 후 냉매압력 하강 추가 냉매량 계산 필요 -8 냉매 충전일정
REF-02 충전포트 누설 이력 실외기 서비스포트/충전밸브 패킹 손상
★ 2020년식 실제 발생 이력		 1순위 누설 경로 -10 포트 점검패킹 준비
REF-03 내부 밸브 패킹 누설 실외기 내부 밸브 패킹/접속부 누설 2순위 누설 경로 -12 긴급 점검부품 준비
REF-04 공사 후 밸브 주변 누설 작업 직후 밸브 주변 (용접 열손상 가능성) 공사 후 48h 내 확인 -10 누설 점검
REF-05 냉매 누설 확정 누설 감지기 또는 압력 저하로 확정 즉시 가동 중단 검토 -15~25 즉시 조치보고서
REF-06 냉매 반복 보충 2회↑ 보충 이력 — 누설 가능성 높음
하절기 ×1.5 가중치 적용		 구조적 누설 조사 필요 -10 누설 조사정밀 점검
REF-07 냉방 성능 민원 + 냉매 부족 입주자 냉방 불만 + 냉매압력 이상 민원 가중치 추가 -12 긴급 충전민원 처리
⚙️ OPR · 운전 이벤트
압축기 운전시간, 고/저압 이상, 성능저하 등 실제 운전 상태 이벤트. DMS 엑셀에서 직접 추출 가능한 항목.
OPR
Operation
운전 시간 · 압력 · 성능 · 부하 관련 이벤트
코드이벤트명발생 조건현장 메모점수 delta추천 액션
OPR-01 운전시간 기준 초과 압축기 누적 운전시간 6,000h 이상
6,000h↑: -5pt8,000h↑: -10pt		 노후화 지표 -5~10 모니터링예방점검
OPR-02 고압 경고 고압 스위치 동작 / 고압 에러코드 발생
반복 3회↑: -10pt 추가		 청소 불량 or 냉매 과충전 확인 -8~18 현장 점검반복 시 긴급
OPR-03 저압 경고 저압 스위치 동작 / 저압 에러코드 냉매 부족 연계 확인 -8 냉매 점검
OPR-04 냉방 성능 저하 설정온도 미달 지속 (2h↑) 감지 필터 막힘 or 냉매 부족 -5 현장 점검모니터링
OPR-05 제상 이상 제상 사이클 비정상 / 동절기 제상 실패 동절기 집중 모니터링 -5 동절기 점검
OPR-06 피크 부하 초과 DMS 피크 제어 한계 도달 / 전력 임계 초과 DMS 2.5 피크제어 설정 검토 -3 피크 모니터링설정 조정
🔧 MNT · 정비 이벤트
점검 수행 여부, 부품 교체, 예방정비 완료 등. 이 카테고리는 가점(+) 이벤트가 포함되는 유일한 코드군.
MNT
Maintenance
점검 · 정비 · 부품교체 · 청소 · 예방정비 완료 이벤트
코드이벤트명발생 조건현장 메모점수 delta추천 액션
MNT-01 정기점검 완료 월간 정기점검 체크리스트 완료 등록 유지 점수 부여 +3/월 보고서 자동
MNT-02 점검 누락 예정 점검일 +3일 내 미수행 알림 발송 후 감점 -5 일정 재설정
MNT-03 부품 교체 완료 불량 부품 신품 교체 완료 등록 가점 + 재교체 이력 추적 +4 이력 기록

🎯 우선순위 엔진
여러 이벤트 동시 발생 시 우선순위 결정 공식. 위험도 × 민원 × 계절 × 재발 4개 가중치 합산.
우선순위 점수 = 위험도 점수 + 민원 가중치(×1.5) + 계절 가중치(×1.5) + 재발 가중치(×1.3)
가중치 항목 조건 계수 적용 예시
위험도 기본 P1~P5 룰 트리거 점수 ×1.0 REF-05 (-15) → 기본 위험도 15점
민원 가중치 OPS-04 or OPS-05 동시 발생 ×1.5 냉매 이상 + 민원 → 15×1.5 = 22.5점
계절 가중치 하절기(6~8월) REF·OPR 이벤트 ×1.5 하절기 냉매 이상 → 추가 1.5배 우선도
재발 가중치 동일 이벤트 코드 90일 내 2회↑ ×1.3 주소충돌 재발 → 8×1.3 = 10.4점
최대 감점 클램프 단일 사건 최대 감점 -30pt 복합 가중치 누적 시 30점 상한
